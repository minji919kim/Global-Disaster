# Global-Disaster

### This is the data report about 26,000 global disasters occurred from 1900 to 2023. By building the visual report which focuses on monetary loss, the number of casualities, and the frequency of different types of disasters, the users can see the general pattern of global disasters more easily. 

### Python is mainly used for data cleaning, and the rest of steps are done with Excel. The raw dataset is publicly published on EM-DAT offical website.
